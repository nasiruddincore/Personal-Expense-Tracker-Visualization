# 💰 Personal Expense Tracker with Data Visualization

A complete Python-based Personal Expense Tracker that helps users record, analyze, visualize, and manage expenses using data analytics and interactive dashboards.

Built using Python, Pandas, SQLite, Matplotlib, Seaborn, and Streamlit.

---

# 📌 Project Overview

Managing personal finances manually can be difficult and time-consuming. This project automates expense tracking and provides powerful visual insights into spending behavior.

The system allows users to:
- Store expenses
- Analyze spending habits
- Visualize trends
- Generate reports
- Track monthly expenses
- Understand payment method usage

This project is ideal for:
- Students
- Employees
- Freelancers
- Families
- Small businesses

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone YOUR_REPOSITORY_URL
```

---

## 2️⃣ Move into Project Folder

```bash
cd Personal-Expense-Tracker-Visualization
```

---

## 3️⃣ Create Virtual Environment

### Windows

```bash
python -m venv venv
```

### Mac/Linux

```bash
python3 -m venv venv
```

---

## 4️⃣ Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ How to Run the Project

## Run Main Expense Analysis

```bash
python main.py
```

---

## Run Streamlit Dashboard

```bash
streamlit run src/dashboard.py
```
